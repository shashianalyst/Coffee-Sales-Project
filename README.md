# Coffee-Sales-Project
Coffee Sales Analysis ☕📊
Welcome to the Coffee Sales Analysis project! This repository contains a comprehensive SQL-based analysis of data from a data-driven coffee business operating across multiple cities. The goal of this project is to analyze customer behavior, product performance, city-wide demographics, and sales revenues to derive actionable business insights.

📌 Project Overview
This project sets up a relational database for a coffee company named coffie_sales and performs deep-dive data analysis using advanced SQL techniques (Aggregations, Window Functions, Joins, and CTEs). The analysis answers critical business questions, such as tracking product popularity, optimizing operations based on city rent versus sales margins, and identifying target expansion markets based on population demographics.
